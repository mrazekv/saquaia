Analyzing the model

Once we are done modeling the system, we can analyze it. Go the menu bar and select Perform analysis in the Analysis menu. If the analysis was successful, the message Analysis performed successfully will be displayed in the Message panel.

Troubleshooting

If the analysis is not performed successfully, this can have several reasons:

The stoichiometric coefficients are not 1,-1 or 0.

The discretization and the bounds are conflicting.

The initial state does not reflect the actual population number of the species, but the population levels.
