Analyzing the model

Fastest way to run a simulation is the Simulator pane.

The combo box at the top of the pane provides a range of simulation methods implemented in the application. Clicking the Simulate button initiates a simulation. Clicking the Transient analysis! button starts transient analysis, performing a number of simulations that is specified in the edit box at the bottom.

Another way to execute simulations is through the Run tab in the Menu bar, which also offers the option to perform multiple simulations without transient analysis by a single click.

Information about the simulation run, such as progress or simulation time, can be viewed in the information section at the bottom.

Segmental simulations

Since segmental simulations (Segmental SSA and Segmental HYB) use memory as a method for acceleration, the Simulator tab in the Menu bar provides various options for users to manage details of these analyses. Each time a new segmental simulation is run, it could be sped up by the segments already memorized from previous runs. The Simulator tab allows the user to clear this memory or display information about it, as shown below.

Lastly, the tab allows the user to fully configure the segmental analysis by selecting the New simulator option. This action opens a dialog box with detailed, configurable information about the segmental analysis settings.
