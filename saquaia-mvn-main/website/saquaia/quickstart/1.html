Getting started

For a detailed description of SAQuaiA, please refer to the Documentation. This quick start guide doesn't require an extensive understanding of chemical reaction networks (CRNs), but some familiarity with traditional CRN analysis could be helpful.

Requirements

SAQuaiA is written in Java 11 and is not compatible with Java 8. Ensure you have the appropriate Java runtime environment installed on your operating system.

Download

Navigate to the Download section in the navigation bar and download the currently available version of SAQuaiA. The file name will be in the format release_date_saquaia.zip, where release_date corresponds to the software version you are downloading.

Running SAQuaiA

Unzip the downloaded zip file, open the resulting folder, and proceed to the code folder. Execute the following command: java -Xmx16g -jar target/saquaia-jar-with-dependencies.jar -gui. Please be patient, as the GUI might take a few seconds to open.
