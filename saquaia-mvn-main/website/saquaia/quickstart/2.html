CRN & Setting

To introduce this part of the application UI, we have selected the Predator_Prey model, which is preloaded when the application launches.

Open the model

The models are located in the folder data/settings/examples. The selection of a specific model is accomplished through the Settings tab in the Menu bar, by choosing the Open option which initiates the file explorer for selection.

CRN

In this pane, users can change the model's name when clicking on the Edit button. Furthermore, they can modify species and reactions either by double-clicking on a specific item or by using a right-click if they also wish to add or delete objects.

Setting

Same as CRN, user can set a name for the particular Setting, which is defined by the following items:

Abstraction: Defines for each species in the model a set of intervals with their representatives in the format [interval beginning, representative, interval end]. Users can modify the interval bounds or customize the intervals by double-clicking the interval area of the particular species.

Growth factor: The number represents how many times the next interval is larger than the previous one. The exception lies with custom intervals, which are preserved when the growth factor changes.

Initial state: Clicking Edit allows user to set the starting number for each species.

End time: Clicking Edit allows user to set the simulation end time.
