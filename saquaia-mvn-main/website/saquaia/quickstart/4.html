Analysis results

Result overview

The Results pane provides an overview of the simulation and transient analysis results. Double-clicking on a specific result displays its graph in a separate dialog window. Right-clicking presents options such as editing the result name, deletion, loading the settings used to calculate the result, and comparison, which will be discussed later.

Simulation result window

Double-clicking on a specific result opens the Result dialog window, which visualizes the evolution of the species count over time.

The dialog's graph allows users to display either species numbers or abstract states (levels). Additionally, users can zoom in on a specific area of the graph through click-and-drag selection towards the bottom-right; selecting in the opposite direction will zoom out to the default view.

The same dialog is used when comparing results from different simulations. Selection is achieved by clicking with the mouse while holding down the Control key. Comparisons can be made using the 'Compare' option in the Menu bar or via the options available when right-clicking in the simulations area. Multiple simulations can have different settings. For comparison purposes, users can select a specific setting from the combobox located beneath the graph.

Transient analysis result window

This dialog is similar to the simulation result dialog, except it displays the result of transient analysis. Note that each species is represented by a unique color, and the species whose color extends to the top of the level possesses a larger mass.

To quantify the difference, users can view the Earth Mover's Distance of the distributions by selecting the checkbox located beneath the graph area.

Speed comparison of transient analyses

The effectiveness of segmental simulations can be observed with the increasing number of simulations, as the memorized segments are reused more frequently. Users can observe the speedup factor by selecting the transient analyses and comparing them, either by using the 'Compare' option in the Menu bar or via the options that appear when right-clicking the transient analysis result area.
