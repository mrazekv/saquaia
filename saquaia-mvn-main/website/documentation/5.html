Before you can start with the visualization, make sure you have performed an analysis.

Setting a time bound

Go to the Simulation menu in the menu bar and click Set time bound. A dialog prompts you to enter a time bound.

Running a simulation

Option 1: Go to the Simulation menu in the menu bar and click Run simulation.

Option 2: Go to the Simulation panel and click Run simulation.

Plotting a simulation

Go to the Simulation panel and select a simulation from the list. Select a species you want to monitor. Click Plot trajectory to plot the trajectory of the selected simulation or Plot distribution to plot the distribution of the selected simulation, i.e. distribution of time spent in the different population levels.

Plotting multiple trajectories You can select multiple simulations from the list and click Plot trajectory to plot them.
